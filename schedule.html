Please use #super-guidebookupdates

The Uber Schedule has already been exported for this year, and further updates should be requested in slack.

Please bear the following in mind:

Every request should include:

  • The name of the panel, guest, or item exactly as it should appear in Guidebook
  • Start time in 12-hour Eastern Timezone
  • End time in 12-hour Eastern Timezone
  • Calendar date of the event (not just the weekday)
  • Gaylord room name
  • Track(s), if applicable
  • Guest/Artist linkage, if applicable

Please do not include extraneous copypasta text (i.e. Description: XXXXX) as we are copy-pasting exactly what you give us. We are not proof-reading or editing to remove this stuff.

Please do not include stories about why things are changes, etc. Requests should be concise, and literal.
